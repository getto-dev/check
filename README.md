# СантехСчёт

PWA-калькулятор для расчёта смет на сантехнические работы.

👉 https://getto-dev.github.io/check/

## Возможности

- 📋 200+ услуг в 12 категориях
- 💰 Автоматический расчёт сметы
- 💸 Учёт скидок
- 📄 Экспорт в PDF для печати
- ✏️ Ручное добавление позиций
- 💾 **Сохранение и открытие смет** — вернитесь к расчёту в любой момент
- 📱 Устанавливается как приложение (PWA)
- 🔌 Работает офлайн

## Сохранённые сметы

В разделе «Смета» доступны две новые кнопки:

- **Сохранить** — сохраняет текущие позиции, адрес и скидку под выбранным
  названием. Можно перезаписать уже существующую смету или создать новую.
- **Открыть** — показывает список всех сохранённых смет (название, дата
  изменения, количество позиций, итоговая сумма). Нажмите «Открыть», чтобы
  загрузить смету и продолжить редактирование. Также доступно переименование
  и удаление.

Все сметы хранятся локально в браузере (localStorage) и не передаются на сервер.
