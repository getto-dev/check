# СантехСчёт

PWA-калькулятор для расчёта смет на сантехнические работы.

## Демо

👉 https://getto-dev.github.io/check/

## Возможности

- 📋 200+ услуг в 12 категориях
- 💰 Автоматический расчёт сметы
- 💸 Учёт скидок
- 📄 Экспорт в HTML для печати
- 📥 Импорт/экспорт смет
- ✏️ Ручное добавление позиций
- 🌙 Тёмная тема
- 📱 Устанавливается как приложение (PWA)
- 🔌 Работает офлайн

## Категории услуг

- 🔥 Отопление
- 🌡️ Тёплый пол
- 💧 Водоснабжение
- 🏠 Котельные
- 💨 Дымоходы
- 🚿 Канализация
- 🔧 Прокладка труб
- 💎 Водоочистка
- ⚙️ Автоматика
- 🔨 Штробление
- 🛠️ Сервис
- 🚽 Сантехника

## Разработка

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
```

## Данные

Прайс-лист хранится в [price-data](https://github.com/getto-dev/price-data) репозитории.
