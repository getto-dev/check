# СантехСчёт

PWA-калькулятор для расчёта смет на сантехнические работы.

👉 https://getto-dev.github.io/check/

## Возможности

- 📋 200+ услуг в 12 категориях
- 💰 Автоматический расчёт сметы
- 💸 Учёт скидок
- 📄 Экспорт в PDF для печати
- ✏️ Ручное добавление позиций
- 💾 **Сохранение и открытие смет в файл** — смета сохраняется локально
  на устройство в виде HTML-файла и открывается обратно для редактирования
- 📱 Устанавливается как приложение (PWA)
- 🔌 Работает офлайн

## Сохранённые сметы (файлы на устройстве)

В разделе «Смета» доступны следующие кнопки:

- **Сохранить** — скачивает текущую смету (позиции + адрес + скидку) в виде
  HTML-файла на устройство. Файл самодостаточный: его можно открыть в любом
  браузере и увидеть красивую читаемую версию сметы, а приложение СантехСчёт
  умеет разобрать встроенные в файл данные и восстановить смету для
  редактирования.
- **Открыть** — открывает диалог выбора файла. Выберите сохранённый ранее
  HTML-файл сметы, и его содержимое загрузится в приложение для
  дальнейшего редактирования. Если в текущей смете уже есть позиции,
  приложение попросит подтвердить замену.
- **PDF** — экспорт текущей сметы в PDF (как и раньше, для печати/отправки).
- **Очистить** — удаляет все позиции текущей сметы.

### Особенности формата

- Файл сохраняется с именем вида `Smeta_<название>_<дата>.html`.
- Внутри HTML встроен JSON-блок `<script type="application/json" id="estimate-data">`
  с полными данными сметы — он используется только при открытии файла
  в приложении.
- Если открыть HTML-файл напрямую в браузере, видна удобочитаемая
  человекопонятная версия сметы (без возможности редактирования).
